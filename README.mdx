# Faster CMS Plugin Development Documentation

This is a Mintlify-powered documentation site for building plugins for Faster CMS.

## Getting Started

### Installation

1. Install Mintlify CLI:
```bash
npm install -g mintlify
```

2. Navigate to the docs directory:
```bash
cd docs/mintlify
```

3. Start the local dev server:
```bash
mintlify dev
```

The documentation will be available at `http://localhost:3000`

## Deployment

### Deploy to Mintlify

1. Create a Mintlify account at https://mintlify.com
2. Connect your GitHub repository
3. Configure the build settings to point to `docs/mintlify`
4. Push your changes to trigger deployment

### Custom Domain

Configure your custom domain in the Mintlify dashboard settings.

## Contributing

To add new documentation:

1. Create a new `.mdx` file in the appropriate directory
2. Add the page to `docs.json` navigation
3. Use Mintlify components (Cards, Accordions, CodeGroups, etc.)
4. Test locally with `mintlify dev`
5. Commit and push

## Mintlify Components

Available components:
- `<Card>` - Cards for highlighting content
- `<CardGroup>` - Group multiple cards
- `<Accordion>` - Collapsible content sections
- `<AccordionGroup>` - Group accordions
- `<Steps>` - Step-by-step instructions
- `<Tabs>` - Tabbed content
- `<CodeGroup>` - Multiple code examples
- `<Note>`, `<Tip>`, `<Warning>`, `<Info>`, `<Check>` - Callout boxes
