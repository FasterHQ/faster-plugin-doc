## What You'll Build

Faster CMS plugins extend the platform with:

<AccordionGroup>
  <Accordion title="Custom Data Models" icon="database">
    Create your own database tables, models, and relationships using Bookshelf.js ORM with full support for migrations and schema versioning.
  </Accordion>

  <Accordion title="RESTful APIs" icon="cloud">
    Build public, member-authenticated, and admin-protected API endpoints with automatic routing and middleware support.
  </Accordion>

  <Accordion title="Admin Interface" icon="desktop">
    Design beautiful admin panels using React/TypeScript with pre-built components for forms, tables, and data visualization.
  </Accordion>

  <Accordion title="Frontend Pages" icon="browser">
    Create user-facing pages with Handlebars templates that integrate seamlessly with your theme.
  </Accordion>

  <Accordion title="Business Logic Services" icon="gears">
    Organize complex workflows with service classes, repositories, and the provider pattern for extensibility.
  </Accordion>

  <Accordion title="Custom Middleware" icon="filter">
    Process requests with custom middleware for authentication, validation, logging, and more.
  </Accordion>
</AccordionGroup>
